# Law Firm Mobile Assistant PWA

تطبيق ويب تقدمي (PWA) مصمم خصيصاً لشاشات الهواتف لإدارة القضايا والمهام وتقديم نظرة سريعة للعميل.

## المزايا الرئيسية

- **لوحة تحكم محمولة** تعرض إحصاءات القضايا والمهام والجلسات القادمة.
- **قائمة القضايا** مع بحث سريع وفلاتر للحالة، مع عرض مهيأ للّمس.
- **قائمة المهام** منظمة حسب الحالة والأولوية مع إمكانية الفرز السريع.
- **إنشاء ومتابعة المهام** عبر زر عائم وخيارات تحكم سريعة لتغيير الحالة.
- **سجل العملاء** يبرز بيانات التواصل والقنوات المفضلة.
- **صفحة تفاصيل القضية** تعرض ملخص الحالة والمواعيد والمهام المرتبطة مع تنقل سلس.
- **تجربة مستخدم متكاملة** باستخدام `Tailwind CSS` وواجهة عربية RTL بالكامل.
- **وضع نهاري/ليلي** مع تبديل لحظي وتذكر التفضيلات عبر الجلسات.
- **تكامل مع الـ API** عبر نفس طبقة الخدمات المستخدمة في التطبيق المكتبي مع بيانات احتياطية للعمل دون اتصال.
- **دعم PWA** بواسطة `vite-plugin-pwa` لتفعيل التثبيت والعمل دون اتصال.

## هيكل المشروع

```
مشروع الجوال/
├── src/
│   ├── pages/        # الشاشات الرئيسية (الرئيسية، القضايا، المهام، العملاء، تسجيل الدخول)
│   ├── components/   # مكونات واجهة قابلة لإعادة الاستخدام (التنقل السفلي، بطاقات، إلخ)
│   ├── contexts/     # سياق المصادقة ومشاركة حالة المستخدم
│   ├── services/     # واجهات لاستدعاء الـ API (قضايا، مهام، مستخدمين، مصادقة)
│   ├── data/         # بيانات احتياطية للاستخدام خارج الاتصال
│   ├── styles/       # ملفات Tailwind وتهيئة المظهر العام
│   └── utils/        # عميل الـ API المشترك
├── public/           # الأصول الثابتة (favicon، manifest يتم توليده أوتوماتيكياً)
├── vite.config.ts    # تهيئة Vite مع دعم PWA
├── tailwind.config.ts
└── package.json
```

## تهيئة البيئة

- يعتمد المشروع على نفس الـ API المستخدم في التطبيق المكتبي عند العنوان `http://127.0.0.1:8000/api/v1`.
- يتم تخزين رمز المصادقة في `localStorage` وإعادة استخدامه عند تشغيل التطبيق المحمول.

## ملاحظات إضافية

- تم تشغيل `npm run build` للتأكد من سلامة المشروع.
- يوجد عدد من التحذيرات المتوسطة من `npm audit` يمكن معالجتها لاحقاً عند ترقية الحزم (غير حرجة حالياً).
- يمكن تكييف الواجهات بسهولة لإضافة صفحات جديدة مثل الوثائق أو الإشعارات باستخدام نفس بنية المكونات.
